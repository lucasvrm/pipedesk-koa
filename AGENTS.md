## Cursor

### Configuração de Regras do Cursor

As regras específicas do Cursor estão em `.cursorrules`. Este arquivo define:

- Diretrizes de estilo e arquitetura
- Padrões de nomenclatura
- Convenções de código
- Práticas recomendadas para o projeto

### Como usar

1. O Cursor carrega automaticamente `.cursorrules` quando você abre o projeto
2. Essas regras influenciam as sugestões e o comportamento do AI pair programmer
3. Mantenha `.cursorrules` atualizado conforme o projeto evolui
