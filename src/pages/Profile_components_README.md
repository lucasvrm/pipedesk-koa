# Profile Components

This directory will contain modular components for the Profile page.

## Planned Components

- ProfileHeader.tsx - Banner and avatar section
- ProfileSidebarInfo.tsx - Sidebar with user info
- ProfileTabs.tsx - Tab navigation component
- EditableField.tsx - Reusable editable field
- CollapsibleSection.tsx - Collapsible sections

## Tabs Directory

The `tabs/` subdirectory will contain individual tab content components:
- TabOverview.tsx
- TabDocuments.tsx
- TabFinancial.tsx
- TabSecurity.tsx
- TabActivity.tsx
