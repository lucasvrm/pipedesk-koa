# Task Management System - User Guide

## Overview

The Task Management System provides a centralized hub for managing all tasks across the DCM platform. It offers powerful filtering, search, and visualization capabilities to help teams stay organized and productive.

## Features

### 1. Centralized Task View
- Access all tasks from a single, unified interface
- Switch between List and Kanban view modes
- Real-time statistics dashboard showing key metrics

### 2. Advanced Filtering

**Filter by Status:**
- **All Tasks** - View every task in the system
- **My Tasks** - Tasks assigned to you
- **Overdue** - Tasks past their due date
- **Today** - Tasks due today
- **This Week** - Tasks due within the next 7 days
- **Milestones** - Important milestone tasks
- **Completed** - Finished tasks

**Filter by Player:**
- Select a specific player track to see only its tasks
- Or view tasks across all players

### 3. Sorting Options
- **Due Date** - Sort by when tasks are due
- **Priority** - Sort by milestone status and overdue status
- **Created** - Sort by creation date (newest first)
- **Updated** - Sort by last update time
- **Alphabetical** - Sort by task title A-Z

### 4. Search
- Real-time search across task titles and descriptions
- Instantly filter results as you type

### 5. Statistics Dashboard

The dashboard provides at-a-glance insights:
- **Total** - All tasks in the system
- **My Tasks** - Tasks assigned to you
- **Overdue** - Tasks that need immediate attention
- **Today** - Tasks due today
- **Completion Rate** - Percentage of completed tasks

### 7. Kanban View

Tasks are organized into four columns:
- **To Do** - Unassigned tasks
- **In Progress** - Tasks with assignees
- **Blocked** - Tasks waiting on dependencies
- **Completed** - Finished tasks

Each card shows:
- Task title and description
- Player track name
- Due date
- Number of assignees
- Number of dependencies
- Milestone flag

### 8. Task Detail Dialog

Click any task to view full details:
- Complete task information
- Status and completion controls
- Player track and deal context
- Due date and creation metadata
- Assignees list
- Dependencies (tasks this task needs)
- Dependents (tasks that need this task)
- Comments section
- Edit and delete actions

**Special Features:**
- Can't complete a task if it has incomplete dependencies (prevents workflow errors)
- Can't delete a task if other tasks depend on it (maintains data integrity)
- Blocked tasks show a warning message

### 9. Mobile Support

The task management view is fully responsive:
- Optimized layouts for mobile screens
- Touch-friendly tap targets
- Smooth scrolling kanban columns
- Accessible navigation

## Usage Tips

### For Individual Contributors
1. Use the **My Tasks** filter to focus on your work
2. Sort by **Due Date** to prioritize what's urgent
3. Check the **Today** filter each morning
4. Use the **Overdue** filter to catch up on delayed work

### For Managers
1. View **All Tasks** to see team workload
2. Filter by specific players to review progress
3. Use **Milestones** filter to track key deliverables
4. Monitor the completion rate metric

### For Workflow Management
1. Use **Blocked** status in Kanban to identify bottlenecks
2. Review dependencies to understand task relationships
3. Create milestones for important checkpoints
4. Assign multiple team members for collaborative tasks

## Keyboard Shortcuts

While in the task view:
- Click on search input to start searching
- Use Tab to navigate between filters
- Space or Enter to select dropdown options

## Integration with Other Features

Tasks integrate seamlessly with:
- **Player Tracks** - Every task belongs to a player
- **Comments** - Discuss tasks with your team
- **Notifications** - Get alerts for task assignments
- **Global Search** - Find tasks from anywhere in the app
- **Activity Log** - Track all task changes

## Best Practices

1. **Set Due Dates** - Helps with prioritization and filtering
2. **Use Milestones** - Mark critical tasks for visibility
3. **Add Dependencies** - Show task relationships clearly
4. **Assign Owners** - Ensure accountability
5. **Write Descriptions** - Provide context for your team
6. **Update Regularly** - Keep task status current
7. **Complete Tasks** - Maintain accurate completion rates

## Common Workflows

### Daily Standup Preparation
1. Filter to **My Tasks**
2. Sort by **Due Date**
3. Review tasks due **Today** and **This Week**
4. Check for **Overdue** items

### Sprint Planning
1. View **All Tasks** for a player track
2. Sort by **Priority**
3. Identify **Milestones**
4. Review **Blocked** tasks in Kanban

### Status Reporting
1. Check the statistics dashboard
2. Review **Completion Rate**
3. Note **Overdue** count
4. Export from Analytics for detailed reports

---

**Navigation:** Access task management from the main navigation bar by clicking the **Tarefas** button (or **Tasks** icon on mobile).
